# 3D Car Drive 🚗💨

A thrilling racing game where you drive on the road, collect coins, and avoid enemies!

## Features 🎮
- 🚗 Smooth car controls with arrow keys
- 🪙 Collect coins to increase your score
- 🚫 Avoid red enemy cars
- 🔄 Play again after game over
- 🎨 Attractive gradient backgrounds and animations

## How to Play 🕹️
1. Click PLAY to start
2. Use left/right arrow keys to move
3. Collect coins, avoid enemies
4. Try to get the highest score!

## Technologies Used 💻
- HTML5 Canvas for graphics
- JavaScript for logic
- CSS for attractive styles

Have fun! 🎉
